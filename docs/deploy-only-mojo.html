wildfly:deploy-only

Full name:

org.wildfly.plugins:wildfly-maven-plugin:3.0.0.Final:deploy-only

Description:

Deploys only the application to the WildFly Application Server without first invoking the the execution of the lifecycle phase 'package' prior to executing itself. If force is set to true, the server is queried to see if the application already exists. If the application already exists, the application is redeployed instead of deployed. If the application does not exist the application is deployed as normal. If force is set to false and the application has already been deployed to the server, an error will occur and the deployment will fail.

Attributes:

  • Requires a Maven project to be executed.
