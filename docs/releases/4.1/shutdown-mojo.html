wildfly:shutdown

Full name:

org.wildfly.plugins:wildfly-maven-plugin:4.1.1.Final:shutdown

Description:

Shuts down a running WildFly Application Server. Can also be used to issue a reload instead of a full shutdown. If a reload is executed the process will wait for the serer to be available before returning.

Attributes:

  • Requires a Maven project to be executed.
